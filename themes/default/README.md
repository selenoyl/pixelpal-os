# Default Theme

This directory is the placeholder for the default PixelPal theme payload.

Expected future contents:

- launcher background art
- boot splash art
- bitmap or TTF fonts
- menu sound effects
- selection highlight assets
- `audio/manifest.toml` plus WAV files for the launcher

The fallback theme direction is intentionally classic 8-bit:

- dark navy background
- saturated sky blue accents
- warm amber highlights
- chunky pixel framing
- no glossy modern UI treatments
